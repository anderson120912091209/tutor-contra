# 專案總覽

## 🎯 產品定位

**家教聲譽平台** 是一個為私人家教打造的「活履歷」系統，提供第三方課程驗證與公開個人檔案展示。

### 核心價值
- 家教缺乏可信的教學能力證明
- 家長難以信任透過社群媒體找到的家教
- 提供跨平台的聲譽身份層（reputation layer）

### 非目標
- **不是**媒合平台
- **不是**課程市集
- 專注於「驗證」與「展示」教學紀錄

## 🏗️ 技術架構

### 前端
- **框架**: Next.js 15 (App Router)
- **語言**: TypeScript
- **樣式**: Tailwind CSS
- **字體**: Noto Sans TC（極簡舒適）

### 後端
- **資料庫**: Supabase (PostgreSQL)
- **認證**: Supabase Auth
- **API**: Next.js Route Handlers
- **部署**: Vercel（建議）

### 安全性
- Row Level Security (RLS) 全面啟用
- 權限分離（家教/家長）
- 伺服器端資料驗證

## 📊 資料模型

### 核心實體
1. **TutorProfile** - 家教檔案
   - 個人資訊、科目、地區
   - 公開網址 slug
   - 教學年資

2. **ParentProfile** - 家長檔案
   - 基本資訊

3. **Student** - 學生
   - 連結家教與家長
   - 科目、年級
   - 活躍狀態

4. **Lesson** - 課程
   - 排程時間
   - 狀態（scheduled/completed/cancelled）

5. **LessonConfirmation** - 課程確認
   - 家教確認、家長確認
   - 最終狀態（verified/disputed/no_show）

6. **Testimonial** - 評價
   - 評分 1-5 星
   - 文字評價
   - 公開/私人

## 🚀 快速開始

### 1. 安裝依賴
```bash
npm install
```

### 2. 設定 Supabase
1. 建立 Supabase 專案
2. 執行 SQL 遷移（`supabase/migrations/`）
3. 設定環境變數（`.env`）

### 3. 啟動開發伺服器
```bash
npm run dev
```

詳細步驟請見 [DEPLOYMENT.md](./DEPLOYMENT.md)

- 直觀展示教學活動
